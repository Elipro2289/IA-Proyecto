# IA-Proyecto
Este proyecto implementa un clasificador de imágenes de gatos y perros utilizando un modelo de Keras. El proceso se divide en varias fases, comenzando por la carga de las librerías necesarias, la activación del modo *legacy* de Keras y la carga del modelo pre-entrenado, que fue **entrenado previamente utilizando Google Teachable Machine**. También se cargan las etiquetas correspondientes a las clases 'Cat' y 'Dogger'. La primera parte del notebook demuestra la inferencia sobre una única imagen de prueba, pre-procesándola (redimensionando, convirtiendo a array NumPy y normalizando) y luego utilizando el modelo para predecir la clase y su probabilidad.
La segunda parte del proyecto extiende esta funcionalidad para evaluar el rendimiento del modelo sobre un conjunto de imágenes de prueba. Se recorre un directorio completo, clasificando cada imagen y comparando la predicción con la etiqueta esperada. Finalmente, se genera un informe que resume la precisión del modelo, la probabilidad media de los aciertos y una lista de las predicciones incorrectas. Es importante destacar que para que este código funcione correctamente en otro entorno, es necesario actualizar las rutas de los archivos (/content/drive/MyDrive/Progra/keras_model.h5, /content/drive/MyDrive/Progra/labels.txt, y /content/drive/MyDrive/Progra/test) a las ubicaciones correctas en tu propio Google Drive o sistema de archivos.
